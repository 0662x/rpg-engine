# RP Content

This directory contains publishable campaign content for RPG Engine.

## Included

- `isekai-farm-campaign-native-v1/`  
  Current native AIGM Kernel V1 campaign package.

## Excluded

Save packages are intentionally not tracked in this public repository.

Excluded local paths include:

- `isekai-farm-save-native-v1/`
- `isekai-farm-save-native-starter-v1/`
- `logs/`
- `archive/`
- `.aigm/`

The campaign package is authoring/runtime content. A save package is a concrete
playthrough state and may contain private game history, SQLite state, audit
events, snapshots, and local runtime artifacts.
